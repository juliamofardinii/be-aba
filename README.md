## Descrição

Este projeto é uma **landing page corporativa** desenvolvida para a clínica **BeAba**.  
O objetivo foi criar uma página moderna, responsiva e funcional, apresentando os serviços de forma clara e com uma interface amigável.  

Participei de todas as etapas: desde a busca de referências, prototipação até o desenvolvimento final, incluindo alinhamento constante com o cliente para garantir o resultado esperado. 🚀  

## 💻 Tecnologias

- **Next.js**: Framework React para aplicações modernas. [Documentação](https://nextjs.org/docs)  
- **React**: Biblioteca JavaScript para construção de interfaces de usuário. [Documentação](https://react.dev/)  
- **TypeScript**: Superset do JavaScript com tipagem estática. [Documentação](https://www.typescriptlang.org/docs/)  
- **TailwindCSS**: Framework CSS utilitário para estilização. [Documentação](https://tailwindcss.com/docs)  
- **ShadcnUI**: Componentes acessíveis e prontos para uso. [Documentação](https://ui.shadcn.com/)  
- **Vercel**: Deploy e hospedagem. [Vercel](https://vercel.com/docs)  

## 🚀 Instruções para Clonar e Executar o Projeto

1. **Clone o repositório**:
    
    ```bash
    git clone <https://github.com/juliamofardinii/react-test.git](https://github.com/juliamofardinii/be-aba>
    ```
    
2. **Acesse o Diretório do Projeto**
    
    ```bash
    cd be-aba
    ```
    
3. **Instale as Dependências**
    
    ```bash
    npm install
    ```
    
4. **Inicie o Servidor de Desenvolvimento**
    
    ```bash
    npm run dev
    ```
